# Super Store Sales Dashboard

## Project Overview

This project presents an interactive **Super Store Sales Dashboard** built using **Power BI** to analyze sales performance, profit, and profitability across regions, categories, customer segments, and products.
The dashboard is designed to provide **high-level KPIs** as well as **detailed breakdowns** to support data-driven business decisions.

---

## Dataset

The dataset contains transactional sales data for a retail superstore, including:

* Order-level sales and profit
* Product categories and sub-categories
* Customer segments
* Regional sales information
* Time-based order details

## Key Insights

* Technology and Office Supplies contribute significantly to overall sales.
* Certain regions outperform others consistently in total sales.
* Not all high-sales categories generate proportionally high profit.
* A small number of products drive a large share of total profit.
